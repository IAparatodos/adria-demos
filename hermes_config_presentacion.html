11 · Gateway

Gateway Conexión del backend

Hermes Desktop arranca su propio gateway local por defecto. Usa uno remoto para controlar un backend en otra máquina o tras un proxy.

Applies to

All profiles · perfiles

Elige a qué perfiles aplica esta conexión. «All profiles» es la conexión por defecto para los que no tienen override.

Ejemplo: da a un perfil concreto su propio host remoto.

Local gateway

por defecto

Arranca un backend privado en localhost. Funciona sin conexión.

Ejemplo: ideal para uso normal y offline en tu propio equipo.

Remote gateway

OAuth / token

Conecta este escritorio a un backend remoto. Los alojados usan OAuth o usuario/contraseña; los self-hosted un token de sesión.

Ejemplo: controla desde tu portátil un Hermes corriendo en un servidor.

Remote URL

https://gateway…

URL base del backend remoto. Admite prefijos de ruta (ej. /hermes).

Ejemplo: pega la URL de tu servidor y pulsa «Test remote».

Save / Test / Diagnostics

Open logs

Prueba el remoto, guarda para el próximo reinicio o guarda y reconecta. «Open logs» abre desktop.log si el gateway falla.

Ejemplo: revisa los logs si la conexión remota no arranca.
